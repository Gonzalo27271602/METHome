¡Hola Gonzalo! Muy bien, gracias, ¿y vos?
Tu pregunta es muy buena y tiene una solución interesante.

🎯 Objetivo:

Querés que una imagen se vea más "alta" (larga verticalmente) del lado izquierdo y más "corta" del lado derecho, pero sin deformarla. O sea, la imagen debe mantenerse en sus proporciones originales, pero visualmente lograr ese efecto.

✅ Posibles soluciones (sin deformar la imagen):

Opción 1: Usar una máscara (mask-image) en CSS

Esto permite "recortar" visualmente una imagen, creando ese efecto de asimetría.

html
<div class="imagen-mascara">
  <img src="tu-imagen.jpg" alt="Descripción">
</div>

css
.imagen-mascara img {
  width: 100%;
  height: auto;
  mask-image: linear-gradient(to right, black 60%, transparent 100%);
  -webkit-mask-image: linear-gradient(to right, black 60%, transparent 100%);
}

🔍 Qué hace esto:
Muestra completamente el lado izquierdo y hace que el lado derecho se desvanezca, como si fuera "más corto".
Podés cambiar 60% por otro valor para ajustar cuánto se ve.

Opción 2: Colocar la imagen en un contenedor con overflow: hidden y modificar la altura

Esto mantiene la imagen normal, pero le "recortás" el lado derecho.

html
<div class="contenedor-img">
  <img src="tu-imagen.jpg" alt="Descripción">
</div>

css
.contenedor-img {
  width: 300px;
  height: 400px;
  overflow: hidden;
  position: relative;
  clip-path: polygon(0 0, 80% 0, 100% 100%, 0% 100%);
}

.contenedor-img img {
  width: 100%;
  height: auto;
  object-fit: cover;
}

🔍 Qué hace esto:
Recorta la imagen en forma diagonal, más alta a la izquierda y más corta a la derecha, sin estirarla ni deformarla.

imagen lado superior ancho 100% lado altura izquierdo 100% lado derecho altura 80%
ChatGPT dijo:

Perfecto Gonzalo, ahora entiendo con más precisión:
Querés que tu imagen ocupe todo el ancho (100%) arriba, y que su altura sea 100% del lado izquierdo y 80% del lado derecho, generando una especie de "diagonal" de izquierda a derecha en el borde inferior.

🎯 Solución ideal: clip-path

Podés usar clip-path para "recortar" visualmente la imagen con una forma personalizada sin deformarla.

✅ Ejemplo completo

html
<div class="imagen-diagonal">
  <img src="tu-imagen.jpg" alt="Mi imagen">
</div>

css
.imagen-diagonal {
  width: 100%;
  height: 400px; /* Ajustalo a lo que necesites */
  overflow: hidden;
}

.imagen-diagonal img {
  width: 100%;
  height: 100%;
  object-fit: cover;
  clip-path: polygon(0 0, 100% 0, 100% 80%, 0% 100%);
}

🔍 ¿Qué hace esto?

  • clip-path: polygon(...) crea una forma de 4 puntos:

    • (0 0): esquina superior izquierda

    • (100% 0): esquina superior derecha

    • (100% 80%): lado derecho, 80% de altura

    • (0 100%): lado izquierdo, 100% de altura

🔺 Resultado: la imagen mantiene su forma, se recorta en una diagonal suave desde la izquierda (100%) hasta la derecha (80%).

📌 Tip adicional:

Si querés aplicar sombra o bordes redondeados a esta imagen recortada, agregalos al contenedor .imagen-diagonal, no a la img, para que no se vean "cortados".
